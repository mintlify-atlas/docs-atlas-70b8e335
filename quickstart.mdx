The Taylor Swift API is a public REST API. There's no SDK to install, no API key to generate, and no account to create. You only need a way to make HTTP requests.

**Base URL:** `https://taylor-swift-api.sarbo.workers.dev`

<Steps>

<Step title="Make your first request">

Run this command to fetch all albums:

<CodeGroup>

```bash curl
curl https://taylor-swift-api.sarbo.workers.dev/albums
```

```javascript JavaScript
const response = await fetch('https://taylor-swift-api.sarbo.workers.dev/albums');
const albums = await response.json();
console.log(albums);
```

```python Python
import requests

response = requests.get('https://taylor-swift-api.sarbo.workers.dev/albums')
albums = response.json()
print(albums)
```

</CodeGroup>

You'll get back a JSON array of every album in the catalog:

```json
[
  {
    "album_id": 1,
    "title": "1989",
    "release_date": "2014-10-27"
  },
  {
    "album_id": 2,
    "title": "Taylor Swift",
    "release_date": "2006-10-24"
  },
  {
    "album_id": 3,
    "title": "Fearless",
    "release_date": "2008-11-11"
  },
  {
    "album_id": 4,
    "title": "Speak Now",
    "release_date": "2010-10-25"
  }
]
```

<Tip>Each album has an `album_id` you'll use to fetch its songs in the next step.</Tip>

</Step>

<Step title="Get songs for an album">

Use an `album_id` from the previous response to fetch all songs on that album. For example, to get the songs on *1989* (album ID `1`):

<CodeGroup>

```bash curl
curl https://taylor-swift-api.sarbo.workers.dev/albums/1
```

```javascript JavaScript
const albumId = 1; // 1989
const response = await fetch(`https://taylor-swift-api.sarbo.workers.dev/albums/${albumId}`);
const songs = await response.json();
console.log(songs);
```

```python Python
import requests

album_id = 1  # 1989
response = requests.get(f'https://taylor-swift-api.sarbo.workers.dev/albums/{album_id}')
songs = response.json()
print(songs)
```

</CodeGroup>

The response is an array of songs on that album:

```json
[
  {
    "song_id": 1,
    "title": "Welcome To New York",
    "album_id": 1
  },
  {
    "song_id": 2,
    "title": "Blank Space",
    "album_id": 1
  },
  {
    "song_id": 3,
    "title": "Style",
    "album_id": 1
  },
  {
    "song_id": 4,
    "title": "Out Of The Woods",
    "album_id": 1
  }
]
```

<Note>Each song has a `song_id` you'll use to fetch its full lyrics.</Note>

</Step>

<Step title="Fetch lyrics for a song">

Use a `song_id` to retrieve the complete lyrics for that song. For example, to get the lyrics for *Blank Space* (song ID `2`):

<CodeGroup>

```bash curl
curl https://taylor-swift-api.sarbo.workers.dev/lyrics/2
```

```javascript JavaScript
const songId = 2; // Blank Space
const response = await fetch(`https://taylor-swift-api.sarbo.workers.dev/lyrics/${songId}`);
const data = await response.json();
console.log(data.lyrics);
```

```python Python
import requests

song_id = 2  # Blank Space
response = requests.get(f'https://taylor-swift-api.sarbo.workers.dev/lyrics/{song_id}')
data = response.json()
print(data['lyrics'])
```

</CodeGroup>

The response contains the song title and full lyrics as a single string:

```json
{
  "song_id": 2,
  "song_title": "Blank Space",
  "lyrics": "Nice to meet you, where you been?\nI could show you incredible things\nMagic, madness, heaven, sin\nSaw you there and I thought\nOh my God, look at that face\nYou look like my next mistake\nLove's a game, wanna play?\n..."
}
```

</Step>

<Step title="Try the random lyrics endpoint (optional)">

You can fetch a set number of random lyric paragraphs — useful for placeholder text or sampling the catalog without targeting a specific song:

<CodeGroup>

```bash curl
curl "https://taylor-swift-api.sarbo.workers.dev/lyrics?numberOfParagraphs=3&shouldRandomizeLyrics=true"
```

```javascript JavaScript
const params = new URLSearchParams({
  numberOfParagraphs: 3,
  shouldRandomizeLyrics: true,
});

const response = await fetch(
  `https://taylor-swift-api.sarbo.workers.dev/lyrics?${params}`
);
const data = await response.json();
console.log(data.lyrics);
```

```python Python
import requests

params = {
    'numberOfParagraphs': 3,
    'shouldRandomizeLyrics': True,
}

response = requests.get(
    'https://taylor-swift-api.sarbo.workers.dev/lyrics',
    params=params,
)
data = response.json()
print(data['lyrics'])
```

</CodeGroup>

```json
{
  "song_id": 47,
  "song_title": "All Too Well",
  "lyrics": "I walked through the door with you\nThe air was cold but something 'bout it felt like home somehow\n\nAnd I left my scarf there at your sister's house\nAnd you've still got it in your drawer even now\n\nOh, your sweet disposition and my wide-eyed gaze\nWe're singing in the car, getting lost upstate"
}
```

<Tip>Set `shouldRandomizeLyrics=false` to get the first N paragraphs of lyrics from the same song rather than randomly selected paragraphs.</Tip>

</Step>

</Steps>

## All endpoints at a glance

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/albums` | List all albums |
| `GET` | `/albums/{albumID}` | List songs on a specific album |
| `GET` | `/songs` | List all songs |
| `GET` | `/songs/{songID}` | Get a single song |
| `GET` | `/lyrics/{songID}` | Get full lyrics for a song |
| `GET` | `/lyrics?numberOfParagraphs={n}&shouldRandomizeLyrics={bool}` | Get N paragraphs of lyrics |
