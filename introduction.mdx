# Taylor Swift API

The Taylor Swift API gives you programmatic access to Taylor Swift's full discography, including albums, individual songs, and complete lyrics. It's free to use, requires no authentication, and handles over 1,000 requests daily — peaking at 170,000 requests per day.

The API is hosted on [Cloudflare Workers](https://workers.cloudflare.com/) with data stored in [Cloudflare D1](https://developers.cloudflare.com/d1/).

## Base URL

```
https://taylor-swift-api.sarbo.workers.dev
```

## Key features

- **No authentication required** — All endpoints are open and publicly accessible
- **JSON responses** — All responses return structured JSON
- **High availability** — Served globally via Cloudflare's edge network
- **Complete discography** — All albums and songs including vault tracks and bonus editions

## Quick example

Fetch all albums with a single `curl` command:

```bash
curl https://taylor-swift-api.sarbo.workers.dev/albums
```

```json
[
  {
    "album_id": 1,
    "title": "1989",
```
