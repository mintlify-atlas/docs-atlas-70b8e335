---
title: 'Get songs in an album'
description: 'Returns all songs belonging to a specific album, identified by its album ID.'
api: 'GET https://taylor-swift-api.sarbo.workers.dev/albums/{albumID}'
---

Returns an array of songs that belong to the specified album.

## Path parameters

<ParamField path="albumID" type="integer" required>
  The unique identifier of the album. Use the [Get all albums](/api-reference/albums/get-all-albums) endpoint to look up valid album IDs.
</ParamField>

## Response

<ResponseField name="[]" type="object[]">
  Array of song objects belonging to the album.

  <Expandable title="properties" defaultOpen>
    <ResponseField name="song_id" type="integer" required>
      Unique identifier for the song.
    </ResponseField>

    <ResponseField name="title" type="string" required>
      Title of the song.
    </ResponseField>

    <ResponseField name="album_id" type="integer" required>
      ID of the album this song belongs to. Matches the `albumID` path parameter.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://taylor-swift-api.sarbo.workers.dev/albums/10
```

</RequestExample>
