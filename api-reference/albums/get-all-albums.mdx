---
title: 'Get all albums'
description: 'Returns a list of all Taylor Swift albums with their IDs and release dates.'
api: 'GET https://taylor-swift-api.sarbo.workers.dev/albums'
---

Returns an array of all albums. Each album includes an ID, title, and release date.

## Response

<ResponseField name="[]" type="object[]">
  Array of album objects.

  <Expandable title="properties" defaultOpen>
    <ResponseField name="album_id" type="integer" required>
      Unique identifier for the album.
    </ResponseField>

    <ResponseField name="title" type="string" required>
      Title of the album.
    </ResponseField>

    <ResponseField name="release_date" type="date" required>
      Release date of the album in `YYYY-MM-DD` format.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://taylor-swift-api.sarbo.workers.dev/albums
```

</RequestExample>

<ResponseExample>

```json 200
[
  {
    "album_id": 1,
    "title": "1989",
    "release_date": "2014-10-27"
  },
  {
    "album_id": 2,
    "title": "Taylor Swift",
    "release_date": "2006-10-24"
  }
]
```

</ResponseExample>
